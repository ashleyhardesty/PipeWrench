This pipe creates a formatted, 4-column base conversion chart that you can use to convert the decimal values 0-255 to their base-2, 8 and 16 equivalents. Run it against a single asterisk (*) as input text.

In Linux, to run this demo from a terminal, do the following:

   - Open a terminal.
   - Change to this folder.
   - Run this shell command:

     echo "*" | pcl "call './Create Base Conv Chart.pip'"
